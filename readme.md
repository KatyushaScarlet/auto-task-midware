# Auto Task Midware

## Install dependency
```
npm install
```

## Run as API provider
```
npm run start
```

## Config example
Replace `<your-api-key>` with your actual API KEY.
```
preference:
    geetest_url: http://127.0.0.1:3000/geetest
    geetest_params: {}
    geetest_data:
        gt: '{gt}'
        challenge: '{challenge}'
        appkey: <your-api-key>
```
